# Productization: Orchestrate
Managed [Dagster](https://github.com/dagster-io/dagster) for orchestrating and executing [dbt](https://github.com/dbt-labs/dbt-core) transformations on mytiki.com data lakes. Simplified execution of pipelines (cleaning, de-identification, unification, etc.) to prepare data for monetization.
